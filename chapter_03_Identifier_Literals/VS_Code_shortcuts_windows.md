# VS Code Default Keyboard Shortcuts for Windows

## General
| Action | Shortcut |
|--------|----------|
| Command Palette | `Ctrl+Shift+P` or `F1` |
| Quick Open (File) | `Ctrl+P` |
| Settings | `Ctrl+,` |
| Keyboard Shortcuts | `Ctrl+K Ctrl+S` |
| Toggle Terminal | `Ctrl+`` |

## Basic Editing
| Action | Shortcut |
|--------|----------|
| Cut line | `Ctrl+X` |
| Copy line | `Ctrl+C` |
| Move line up/down | `Alt+↑` / `Alt+↓` |
| Copy line up/down | `Shift+Alt+↑` / `Shift+Alt+↓` |
| Delete line | `Ctrl+Shift+K` |
| Insert line below | `Ctrl+Enter` |
| Insert line above | `Ctrl+Shift+Enter` |
| Indent line | `Ctrl+]` |
| Outdent line | `Ctrl+[` |
| Jump to matching bracket | `Ctrl+Shift+\` |
| Toggle line comment | `Ctrl+/` |
| Toggle block comment | `Shift+Alt+A` |
| Toggle word wrap | `Alt+Z` |

## Navigation
| Action | Shortcut |
|--------|----------|
| Go to line | `Ctrl+G` |
| Go to file | `Ctrl+P` |
| Go to symbol in file | `Ctrl+Shift+O` |
| Go to symbol in workspace | `Ctrl+T` |
| Go to definition | `F12` |
| Peek definition | `Alt+F12` |
| Go back | `Alt+←` |
| Go forward | `Alt+→` |
| Go to beginning of line | `Home` |
| Go to end of line | `End` |
| Go to beginning of file | `Ctrl+Home` |
| Go to end of file | `Ctrl+End` |
| Scroll line up/down | `Ctrl+↑` / `Ctrl+↓` |
| Scroll page up/down | `Alt+PgUp` / `Alt+PgDn` |
| Switch between open tabs | `Ctrl+Tab` |

## Search & Replace
| Action | Shortcut |
|--------|----------|
| Find in file | `Ctrl+F` |
| Replace in file | `Ctrl+H` |
| Find in files | `Ctrl+Shift+F` |
| Replace in files | `Ctrl+Shift+H` |
| Toggle search details | `Ctrl+Shift+J` |
| Find next / previous | `F3` / `Shift+F3` |
| Select all occurrences of find match | `Ctrl+Shift+L` |

## Multi-cursor & Selection
| Action | Shortcut |
|--------|----------|
| Add cursor above | `Ctrl+Alt+↑` |
| Add cursor below | `Ctrl+Alt+↓` |
| Select all occurrences | `Ctrl+Shift+L` |
| Add selection to next find match | `Ctrl+D` |
| Column (box) selection | `Shift+Alt` + drag |
| Column select up/down | `Shift+Alt+↑` / `Shift+Alt+↓` |
| Column select left/right | `Shift+Alt+←` / `Shift+Alt+→` |
| Expand/shrink selection | `Shift+Alt+→` / `Shift+Alt+←` |
| Undo cursor position | `Ctrl+U` |
